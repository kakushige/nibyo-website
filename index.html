日本語版

1. はじめに

「nibyo」（以下、本アプリ）は、ユーザーのプライバシーを尊重し、個人情報の取扱いに細心の注意を払います。本ポリシーは、本アプリがどのような情報を取得し、どのように利用するかを説明するものです。

2. 取得する情報

2.1 端末上で処理する情報（外部送信なし）

以下の情報はすべてユーザーの端末内のみで処理され、サーバー等への送信は行いません。

  • カメラ映像 — 動画クリップを撮影するために使用します
  • マイク音声 — ユーザーが設定で「マイクON」を有効にした場合のみ録音されます。デフォルトはオフです
  • 端末内の動画ファイル — Android のフォトピッカー経由で、ユーザーが明示的に選択した動画のみアクセスします
  • アプリの設定・編集状態 — 撮影クリップの一覧、編集中のプロジェクト状態、カメラ設定を端末内のSharedPreferencesに保存します

2.2 外部送信される情報

Google Fonts (Downloadable Fonts)

タイトル機能で書体を表示するため、Android が提供する Google Fonts API を介してフォントファイルを Google のサーバーから取得することがあります。これは Android OS の標準機能であり、本アプリは取得したフォントファイル以外の情報を送受信しません。

Firebase Crashlytics（Google LLC 提供）

本アプリの品質改善のため、アプリがクラッシュまたはエラーを起こした際に、以下の匿名化された診断情報を Google の Firebase サービスへ自動送信します。

  • 端末モデル、製造元、Android バージョン
  • アプリのバージョン、Crashlytics ID（端末固有のランダムID）
  • クラッシュのスタックトレース、エラーメッセージ
  • アプリ内部の状態（例: Media3 ライブラリのバージョン、編集中のクリップ数）

これらの情報には、個人を特定できる情報、ユーザーが撮影・選択した動画ファイルの内容、タイトルテキスト等は含まれません。

Firebase Analytics（Google LLC 提供）

Crashlytics の動作要件として Firebase Analytics SDK が同梱されますが、本アプリ独自のイベント送信は行いません。Analytics による自動収集情報は Firebase のプライバシー設定に従います。

3. 取得しない情報

本アプリは以下を取得しません。

  • 氏名、メールアドレス、電話番号などの個人情報
  • 位置情報
  • 連絡先、カレンダー、SMS等のその他の端末データ
  • 広告ID（Crashlytics の自動収集を除く）
  • ユーザーが撮影・編集した動画の内容そのもの

4. 情報の用途

情報用途
カメラ・マイク・端末内動画アプリの中核機能（撮影・編集・書き出し）の提供
アプリ設定・編集状態編集途中の状態を再起動後に復元するため
Crashlytics の診断情報本アプリの不具合特定と品質改善

5. 情報の第三者提供

本アプリはユーザーの情報を第三者に販売・提供しません。Crashlytics の診断情報は Google LLC が運営する Firebase サービス上で処理されます（Firebase プライバシーポリシー）。

6. データの削除

  • 端末内データはアプリのアンインストール、または Android 設定からのアプリデータ削除でいつでも消去できます
  • Crashlytics で収集された診断情報の削除を希望される場合、下記の連絡先までご連絡ください

7. 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。13歳未満のお子様の情報を意図的に収集することはありません。

8. ポリシーの変更

本ポリシーは変更されることがあります。重要な変更がある場合、アプリ内またはストアページで告知します。

English version

1. Introduction

This Privacy Policy describes how the "nibyo" mobile application ("the App") handles user information.

2. Information We Process

2.1 Information processed on-device only (no transmission)

The following data is processed entirely on the user's device and is not transmitted off-device by the App:

  • Camera footage — used to capture video clips
  • Microphone audio — recorded only when the user explicitly enables the "Microphone" toggle in camera settings (off by default)
  • Videos selected from the device — accessed only through Android's Photo Picker, limited to videos the user explicitly selects
  • App preferences and edit state — clip library, in-progress project state, and camera settings are stored in local SharedPreferences

2.2 Information transmitted to third parties

Google Fonts (Downloadable Fonts)

The title feature uses Android's Google Fonts API, which fetches font files from Google servers. This is a standard Android OS facility; the App does not transmit additional information beyond the standard font request.

Firebase Crashlytics (provided by Google LLC)

To improve app quality, when the App crashes or encounters errors, the following anonymised diagnostic data is automatically sent to Google's Firebase service:

  • Device model, manufacturer, Android OS version
  • App version, Crashlytics installation ID (random per-install identifier)
  • Crash stack trace and error message
  • Internal app state (e.g. Media3 library version, number of clips being edited)

This data does not include any personally identifiable information, the contents of videos you record or import, or any title text you enter.

Firebase Analytics (provided by Google LLC)

Firebase Analytics SDK is included as a dependency of Crashlytics. The App does not send any custom events. Automatic data collection follows Firebase's default privacy settings.

3. Information We Do NOT Collect

  • Name, email address, phone number, or other personal identifiers
  • Location data
  • Contacts, calendar, SMS, or other device data
  • Advertising ID (beyond automatic Firebase Analytics collection)
  • The contents of videos you record or edit

4. Purposes

DataPurpose
Camera, microphone, on-device videosCore app functionality (recording, editing, export)
App preferences and edit stateRestoring in-progress edits after app restart
Crashlytics diagnosticsBug identification and quality improvement

5. Sharing With Third Parties

We do not sell or share your information with third parties. Crashlytics diagnostic data is processed by Google's Firebase service (Firebase Privacy Policy).

6. Data Deletion

  • On-device data can be cleared at any time by uninstalling the App or clearing app data via Android settings
  • To request deletion of Crashlytics data, please contact us at the email below

7. Children's Privacy

The App is not directed at children under the age of 13. We do not knowingly collect information from children under 13.

8. Changes

We may update this Policy. Material changes will be announced in the App or on the store listing.
